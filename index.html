Some Basic Questions and Answers

How does javascript work?

JavaScript is a single threaded client-side high level softly coded programming language.JavaScript has a great functionalities for users and for this reason we can create our website dynamically more responded and intelligent.JavaScript is behave like asynchronous,this means it waiting in a loop for a network request, and accepting more incoming requests while the first one is being handled.JavaScript executed through JavaScript engine. Nowadays most of the browsers have their own JavaScript engine, V8 engine own by Chrome. Using Chrome devTool we can execute JavaScript more specificlly.

what does javascript event loop do?

We know that JavaScript is asynchronous and event loop is the main reason for that.JavaScript language access data through the browser APIs. All the operations are executed through call stack, when any function fully executed it poped from the stack.Then new functions are sending from event queue.Event loop continously check that if any stack is empty or not from queue.In this way event loop continously processed.

Local storage vs Session storage?
Local Storage
  • When browser closed, local storage store data.
  • When user revisit the browser, data will availabe for them.
  • Browser support is for limited time.
  • Local Storage is protected from XSS attacks.
    • Session Storage
  • When browser closed, session storage clean data.
  • When user revisit the browser, data will not availabe for them.
  • Browser support for limited time.
  • Session Storage is not fully protected from XSS attacks.
